    Enum JSRuntimeMode

    The execution mode of the BunitJSInterop.

    Namespace: Bunit
    Assembly: Bunit.Web.dll
    Syntax
    public enum JSRuntimeMode : int

    Fields

    Name Description
    Loose

    Loose configures the BunitJSInterop to return default TValue for calls.
    Strict

    Strict configures the BunitJSInterop to throw an JSRuntimeUnhandledInvocationException exception when a call to for has not been setup.
